# PAIR IT TO WIN IT

[pair it to win it](https://sw33t4sh0n3y.github.io/Pair-it-to-win-it/)


## Game Intructions

The game establishes a creative, fun twist on the classic memory game. A key is 
provided that shows the pairs you're trying to locate. Each match represents a
word, phrase or place. Players take turns flipping two cards at a time, keep the
pairs if they match and continue their turn. If there is no match, the next player 
takes their turn. The game ends when all the matches are found or when 35 turns 
are up, th player with the most pairs wins.

Memory games like this are an fun entertaining way to help strengthen cognitive 
skills by boosting focus, improving recall and training the brain to recognize
patterns, all of which support overall mental sharpness and development.
